
Lendo Developer Assignment
---------------------------------
Google Places API and GUI
https://developers.google.com/places/web-service/intro
Task description
Using Google Places API, create a REST API providing name and street address for bicycle
stores in Stockholm located within a 2 kilometer radius around Sergels torg. Go is our
preferred language but feel free to use the language you feel most comfortable in.
Requirements
Your REST API should serve responses in JSON and be sufficiently tested.
Rate limiting
The Google API is rate limited, make sure you mock your dependencies to be able to test
frequently.
